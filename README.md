# Brasileiras-fodas
*Uma ideia de mulheres que mudaram a forma de pensar no Brasil!*

## 0 que é este projeto?
Este repositório guarda o arquivo do meu projeto final do *Curso CITE*. Eu decidi criar um site com o intuito de apresentar grandes pensadoras brasileiras.

A ideia principal é construir um espaço onde outras meninas possam aprender mais sobre as grandes mulheres do nosso país!

## 0 que tem aqui dentro?
Aqui no nosso repositório do GitHub, você vai encontrar:
* 0 mapa da minha ideia (Project Canvas).
* Os códigos que escrevi.
